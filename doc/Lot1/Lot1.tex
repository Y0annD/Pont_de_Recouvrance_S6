\section*{Objectif}
Le but de ce projet est de modéliser le \textit{Pont national de
  Brest} tel qu'il était de 1861 à 1944.\\
Nous souhaitons permettre la visualisation de ses mécanismes et les
observer en fonctionnement tout en s'assurant de la cohérence de nos modèles.\\
\section*{Fonctionnalités}
Les fonctionnalités prévues pour notre application sont:
\begin{itemize}
\item Mettre en mouvement le pont.
\item Se déplacer sur le pont.
\item Pouvoir consulter l'historique du pont.
\end{itemize}

\section*{Perspectives d'amélioration}
Voici quelques améliorations que nous pourions mettre en place si le
temps nous le permet:
\begin{itemize}
\item Ajout d'intelligences artificielles (IA).
\item Faire passer un bateau sous le pont.
\item Possibilité de vue éclatée.
\item Commander les IA.
\end{itemize}
