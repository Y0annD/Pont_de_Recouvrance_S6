\section*{Objectif}
Préparer les diagrammes UML
\section*{Travail effectué}
Lors de cette séance, nous avons énuméré un ensemble de
fonctionnalités liées aux systèmes:

\section*{Actions}
Mise en mouvement du pont: 
\begin{itemize}
\item Rentrer les verrous
\item Ouvrir les machoires
\item Actionner les cabestants
\end{itemize}

Fermeture:
\begin{itemize}
\item    fermer les machoires
\item    fermer les verrous
\end{itemize}

\section*{Actions de l'acteur}
Les différentes actions que nous avons identifiées sont:
\begin{itemize}
\item Se déplacer
\item Intéragir avec les différents mécanismes
\item Changer la force applicable sur les cabestants
\end{itemize}

\section*{Systèmes globaux identifiés}
Nous avons identifiés 4 systèmes globaux, eux-même constitué de
sous-système (que nous détaillerons plus tard):
\begin{itemize}
\item Verrous
\item Machoires
\item Cabestants et engrenages
\item Travées
\end{itemize}

\section*{Idées abordées}
Nous avons abordé l'idée que l'utilisateur puisse se déplacer
librement, puis lorsqu'il actionne un système, on déplace la camèra
sur ce système, puis à partir de ce moment, il peut continuer à ce
déplacer librement.
